# 🕵️ Stealth Lead Generation Scraper

> **Estimated Market Value: $99** (Data extracting automation scripts)

This plug-and-play toolkit bypasses Cloudflare and DataDome to extract high-quality B2B leads from online directories, complete with CSV export and proxy rotation.

## Features
- **Playwright Extra + Stealth Plugin:** Evades advanced bot-detection out of the box.
- **Proxy Rotation:** Easy setup for residential proxy arrays.
- **Fast-CSV:** Automatically outputs directly into a cleanly formatted `.csv` file for easy import into your CRM (HubSpot, Salesforce, Apollo).

## Usage
1. `npm install playwright-extra puppeteer-extra-plugin-stealth fast-csv`
2. `node scraper.js`

*Deploy this script as a recurring cron job on your VPS to automatically build hyper-targeted email lists around the clock.*
