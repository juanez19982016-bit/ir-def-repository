▶Tech Stack Details

• Frontend: Next.js 15, React 20, TypeScript 6

• Auth: Auth.js v5 with Google, GitHub, Credentials

• Database: PostgreSQL + Prisma ORM

• Payments: Stripe with webhooks

• Email: Resend + React Email

▶API Design Principles

1. All routes use Zod validation

2. Rate limiting on all public endpoints

3. Proper error handling with typed responses

4. Webhook verification for all external services
