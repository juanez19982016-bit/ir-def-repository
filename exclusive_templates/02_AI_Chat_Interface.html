Build me a REST API endpoint in Node.js with Express that handles user registration with email validation and password hashing.

Here's a complete user registration endpoint with bcrypt for password hashing and zod for input validation:
server.js
import express from 'express';
import bcrypt from 'bcrypt';
import { z } from 'zod';

const app = express();
app.use(express.json());

// Validation schema
const registerSchema = z.object({
  email: z.string().email(),
  password: z.string().min(8),
  name: z.string().min(2),
});

app.post('/api/register', async (req, res) => {
  const result = registerSchema.safeParse(req.body);
  if (!result.success) {
    return res.status(400).json({ errors: result.error });
  }
  const hashed = await bcrypt.hash(result.data.password, 12);
  // Save to database...
  res.status(201).json({ message: 'User created' });
});
Key points:
• Uses zod for type-safe validation (email format + password length)
• bcrypt with 12 salt rounds for secure password hashing
• Returns proper HTTP status codes (400 for validation, 201 for success)
