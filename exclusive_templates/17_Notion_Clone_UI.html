💡 Architecture Notes
💡
We're using the @app-router pattern with server components. All data fetching happens on the server, client components are used for interactivity only.
The new architecture uses a layered approach:
Presentation layer (React Server Components)
Business logic layer (Server Actions)
Data access layer (Prisma ORM)
Infrastructure layer (Supabase + Vercel)
